# 示例：通用非技术学习

## 用户输入

```text
调用 Learning Accelerator Skill。我想提升公开演讲能力，平时不写代码，也没有技术背景。
```

## 期望输出风格

## 学习状态

- 学习领域：communication
- 当前目标：提升公开演讲能力
- 已知技能：日常表达
- 薄弱点：结构化表达、临场紧张
- 本轮模式：onboarding / practice

## 先定位

公开演讲不是“天生会不会说话”的问题，而是把想法组织成听众能跟上的结构，并在紧张时仍能稳定表达。

## 最小练习

今天只练一个结构：`观点 - 理由 - 例子 - 收尾`。

题目：

```text
用 60 秒介绍一本你最近看过的书。
```

要求：

- 先说你的观点。
- 给一个理由。
- 举一个具体例子。
- 最后用一句话收尾。

## 评价标准

- 是否在 60 秒内讲完。
- 听众是否能听出你的核心观点。
- 例子是否具体。
- 收尾是否干净。

- 待复习：观点-理由-例子-收尾
- 下一步：录音一次并复盘卡顿点
