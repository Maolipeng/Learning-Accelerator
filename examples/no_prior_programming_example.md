## 先定位

函数可以理解成“有名字的一组步骤”。以后你想重复做这组步骤时，不需要重新写一遍，只要调用这个名字。

比如你每天泡茶，大概有这些步骤：

1. 烧水。
2. 放茶叶。
3. 倒水。
4. 等几分钟。

如果把这组步骤叫做“泡茶”，下次你只要说“泡茶”，就代表执行整套步骤。编程里的函数也是这个思路。

## 最小 Demo

创建 `demo.py`：

```python
def greet(name):
    return "你好，" + name

message = greet("小明")
print(message)
```

运行：

```bash
python demo.py
```

输出：

```text
你好，小明
```

## 代码怎么读

- `def greet(name):`：定义一个叫 `greet` 的函数。
- `name`：函数需要你给它的一个信息。
- `return`：把结果交回去。
- `greet("小明")`：真正使用这个函数。

## 常见坑

- 只写了函数定义，不代表函数已经运行。
- 缩进不能乱改，缩进表示哪些代码属于函数。
- `return` 和 `print` 不一样：`return` 是把结果交回程序，`print` 是显示在屏幕上。
